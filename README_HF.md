# üë§ Live Face Recognition

A real-time face recognition system built with DeepFace and Streamlit.

## Features
- **Enroll Users**: Register faces using webcam
- **Verify Faces**: Compare faces against enrolled users
- **User Management**: View and delete enrolled users

## How to Use
1. Go to **Enroll User** to register a new face
2. Take a photo with your webcam
3. Go to **Verify Face** to check if a person is registered

## Technology
- DeepFace (Facenet512 model)
- Streamlit
- OpenCV
- SQLite
